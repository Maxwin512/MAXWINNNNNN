
## Практическая работа №6: Документация для разворачивания (README)

### 1. Название и назначение
**Личные финансы v1.0** — веб-приложение для учёта доходов и расходов. Позволяет управлять балансом, создавать счета, вести историю операций. Включает авторизацию, валидацию данных, экспорт в CSV и тёмную тему.

### 2. Стек технологий
- **HTML5** — разметка
- **CSS3** — стили, адаптив, тёмная тема
- **Vanilla JS** — логика приложения
- **localStorage** — хранение данных в браузере
- **Нативные API** — экспорт CSV

### 3. Установка
Установите любой современный браузер (Chrome, Firefox, Edge и др.).

```bash
git clone https://github.com/ваш-репозиторий
cd MAXWINNNNNN
```

Проект не требует дополнительных зависимостей.

### 4. Запуск
1. Откройте папку `MAXWINNNNNN`
2. Дважды кликните по `index.html` — приложение откроется в браузере
3. **Тестовый вход**:  
   - Логин: `admin`  
   - Пароль: `Admin123!`
4. Или зарегистрируйте нового пользователя прямо в приложении.

Все данные сохраняются в браузере автоматически.
